影片中 Expo Go APP 的操作流程如下：

1. **點擊專案**：在影片開頭，用戶在 Expo Go 的首頁點擊了名為 `gogocar-mobile` 的專案。
2. **出現選擇畫面**：點擊後，畫面跳轉到該專案的頁面，並**出現了分支（Branches）選擇畫面**，列出了兩個選項：`Branch: production` 和 `Branch: preview`。
3. **點擊 production 分支與錯誤提示**：用戶首先點擊了 `Branch: production`。進入後，畫面顯示了該分支的更新紀錄，但**出現了不相容的錯誤提示**，每個更新項目下方都寫著「Not compatible with this version of Expo Go」（與此版本的 Expo Go 不相容），因此無法開啟。
4. **返回並點擊 preview 分支**：用戶隨後點擊左上角返回上一頁，接著點擊了另一個選項 `Branch: preview`。
5. **最終顯示內容**：點擊 `preview` 分支後，APP 最終顯示的是該分支的**更新紀錄列表**（Updates）。這些列表項目看起來是正常的，沒有顯示不相容的錯誤，並且畫面右上角出現了一個「Open」按鈕，表示可以開啟這個版本的專案。
